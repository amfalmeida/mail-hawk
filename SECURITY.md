# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Use GitHub's [private vulnerability reporting](https://github.com/amfalmeida/mail-hawk/security/advisories/new) feature
3. Provide details: affected versions, steps to reproduce, potential impact

We will respond within 48 hours and provide a timeline for a fix.

## Supported Versions

Onlythe latest release is actively maintained.
